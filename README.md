# Project ServicioSocial

## Features

- [x] Login authentification with Gmail
- [x] Secured API endpoints with token
- [x] Documented API with [Swagger](http://localhost:3000/api-docs/)

## How it works

1. Backend: API built using Node.js that connects to an Atlas MongoDB.
2. Frontend: Designed with Vite React.
3. Testing: Read the [testing plan](TESTING_PLAN.md) for more info.

### Backend Dependencies

**Core Framework & Server:**
- Express.js - Web server framework
- Mongoose - MongoDB database integration
- CORS - Cross-origin request handling

**Authentication & Security:**
- Google Auth Library - Google OAuth integration
- JWT Decode - Token processing

**API & Documentation:**
- Swagger - API documentation and UI

**File Handling:**
- Multer - File upload management

**Development & Production:**
- Nodemon - Development auto-restart
- PM2 - Production process manager
- Dotenv - Environment variables

**Testing:**
- Jest + Supertest - Testing framework

---

### Frontend Dependencies

**Core Framework:**
- React 18 - UI library
- React DOM - DOM rendering

**State & Routing:**
- Recoil - State management
- React Router - Navigation

**UI & Styling:**
- Ant Design - UI component library
- Tailwind CSS - Utility CSS framework
- React Icons - Icon library

**HTTP & Validation:**
- Axios - API requests
- Zod - Schema validation

**Authentication:**
- Google OAuth - Login integration
- JWT Decode - Token handling

**Utilities:**
- Moment - Date manipulation

**Build Tools:**
- Vite - Build tool and dev server
- ESLint - Code linting
- PostCSS - CSS processing

---

## Key Technologies Used
- **Backend:** Node.js + Express + MongoDB
- **Frontend:** React + Vite + TypeScript
- **Styling:** Ant Design + Tailwind CSS
- **Auth:** Google OAuth + JWT
- **State:** Recoil (frontend)
- **Testing:** Jest (backend)
